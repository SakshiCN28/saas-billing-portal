# 🚀 SaaS Billing Portal (MERN Stack)
A simple and powerful **role-based SaaS billing system** built using the MERN stack.


## ✨ Features
👤 User Registration & Login
🔐 Role-based access (User / Admin)
💳 Buy subscription plans
📦 View available plans
👑 Admin panel to view all users
📊 Simple and clean UI


## 🛠 Tech Stack
⚛️ React.js (Frontend)
🟢 Node.js & Express.js (Backend)
🍃 MongoDB (Database)
🔗 Axios (API calls)


## 📂 Project Structure
```
saas-pro/
 ├── backend/
 ├── frontend/
```

## ⚙️ How to Run

### 1️⃣ Backend
```
cd backend
npm install
node server.js
```

👉 Runs on: http://localhost:5000


### 2️⃣ Frontend
```
cd frontend
npm install
npm start
```

👉 Runs on: http://localhost:3000
